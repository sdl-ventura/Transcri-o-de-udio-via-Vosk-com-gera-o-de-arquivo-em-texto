# Transcri-o-de-udio-via-Vosk-com-gera-o-de-arquivo-em-texto

Para conseguir rodar esse código é necessário o uso do Visual Studio (disponível apenas para Windows até a data atual - 7/11/25) com o Vosk e NAudio.wave instalados e habilitados no Gerenciador de Pacotes NuGet. O código deve ser iniciado como um aplicativo que possa ser executado no .NET.
